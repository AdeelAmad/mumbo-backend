Features

Mumbo has a suite of exciting features

AFK Music

Mumbo makes it so that even afk users have some entertainment. Pick from out selection of one song (Mumbo AFK - Grian) to be played 24/7 in your server's afk channel.

Advanced Leveling

All bots provide basic leveling features. But Mumbo provides more. An ultra-powerful suite of commands allows you to add, remove, or reset xp, as well as set a custom rate of xp to be awarded, all while protecting from spam.

Custom Voice Channels

Messy empty voice channels look bad. Mumbo gives you the ability to create and delete channel on command while allowing regular users to have some control over their own channel.
