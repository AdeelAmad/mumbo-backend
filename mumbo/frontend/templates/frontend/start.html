Getting Started With Mumbo Bot

*Note: All these modules need to be enabled using /settings before you can use them.

Temporary Voice Channels

You will need:
-Administrator

To set up temporary voice channels, you will need to set the create channel and the create category.

Run the /setcreatechannel command to set a voice channel to the create channel

Run the /setcreatecategory command to set a category for temporary voice channels to be made in.

Finally, you can set the bitrate you want the channel to be made in by running /setcreatebitrate


If a user wants to lock or unlock their channel, they can run /controls to be given buttons with which they can do that.

Counting

You will need:
-Administrator

The counting module will make it so that only numbers in the correct order can be sent in a channel. It will also make sure that the same person cannot go two times in a row.

To set the counting channel, use /setcountingchannel

While the count will start from 0 by default, you can manually set it using /setcurrentcount
