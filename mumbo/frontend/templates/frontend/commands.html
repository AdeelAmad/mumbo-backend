Commands:

  • /controls

    Access the controls for a voice channel

  • /setcountingchannel

    Sets the channel for the bot to monitor for counting

  • /setcreatebitrate

    Sets the bitrate for voice channels to be created with

  • /setcreatecategory

    Sets the category for voice channels to be created in

  • /setcreatechannel

    Sets the channel for the bot to monitor for voice channel creation

  • /setcurrentcount

    Sets the current count of the counting channel

  • /settings

    Change bot settings for the server

  • /stats

    Bot stats and uptime
